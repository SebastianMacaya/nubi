# Desafio 2

## Instalacion

Instalar las dependencias con NPM O YARN .

```bash

npm i
```

## Iniciar Servidor

```Javascript
npm start

```

## API Reference

#### Traer con propiedad "todolar"

```http
  GET /api/curriencies/
```

#### Traer Default

```http
  GET /api/curriencies/Default
```

```http
http://localhost:8080/api/curriencies/
http://localhost:8080/api/curriencies/default
```
